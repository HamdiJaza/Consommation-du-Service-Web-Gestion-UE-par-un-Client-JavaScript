# Gestion_UE_Student

This project is an educational Java application designed for managing Teaching Units (UE) and Modules for students. It is not a library to be downloaded, but a pedagogical example to illustrate the design of an academic management application.

## Project Objective

The main goal is to provide a foundation for understanding:
- Structuring a Java project with Maven
- Managing entities (Teaching Unit, Module)
- Implementing business logic
- Exposing resources via REST services

## Project Structure

- **entities/**: Contains classes representing the main entities (Module, Teaching Unit).
- **metiers/**: Contains the business logic for managing modules and teaching units.
- **ressources/**: Contains classes exposing REST services and the user interface.
- **utilities/**: Contains utilities needed for the project (e.g., REST activation).
- **resources/**: Folder for additional resources (not detailed here).
- **test/**: Folder intended for unit tests.

## Usage

This project is intended for learning and understanding the structure of a Java application focused on academic management. It can be used as a basis for practical work, demonstrations, or student projects.

## Notes

- This project is not intended for production use.
- It is not a library to be integrated into other projects.
- No specific external dependencies are required beyond those defined in the `pom.xml` file.
